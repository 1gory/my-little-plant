# Исходники семечек (raw)

Сюда клади **высокого разрешения** PNG семечек прямо из нейросети
(≥512px по большей стороне), с именами:

- `radish.png`
- `basil.png`
- `tomato.png`
- `sunflower.png`

Потом запусти из корня проекта:

```
bash tools/normalize-seeds.sh
```

Скрипт обрежет поля и ужмёт все четыре с одним коэффициентом → единая
зернистость. Готовые файлы лягут в `icons/seeds/`. Если исходник меньше
нужного размера — скрипт предупредит (нужно перегенерить крупнее).

Эта папка не используется приложением — только хранилище исходников.
